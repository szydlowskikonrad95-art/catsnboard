# Cats'N'Board — wtyczki i motyw 🐾

**Produkt = dwie wtyczki** do samodzielnego zarządzania stroną kociego pensjonatu
(galeria, wydarzenia z zapisami, polska wersja) + **motyw-nośnik** odtwarzający obecny
wygląd strony klienta. Prostym językiem, bez potrzeby informatyka.

## Co jest w środku

| Element | Opis |
|---------|------|
| **`pnb-blocks/`** | Wtyczka „PNB Galeria i Wydarzenia" — galeria premium + kalendarz wydarzeń z zapisami gości (bloki Gutenberga, edycja klikając w podgląd). |
| **`pnb-auto-pl/`** | Wtyczka „PNB Polska Wersja (AI)" — strona po polsku z przełącznikiem PL/EN; zmiany treści tłumaczą się same po zapisie. |
| **`catsnboard/`** | Motyw WordPress — wygląd strony (hero, sekcje, animacje, osobne podstrony). |
| **`INSTRUKCJA-DLA-KLIENTA.md`** | Instrukcja obsługi krok po kroku. **Zacznij od niej.** |

## Instalacja (skrót — pełna w instrukcji)

1. Motyw: *Wygląd → Motywy → Dodaj → Wyślij* → zip motywu → Włącz
2. Wtyczki: *Wtyczki → Dodaj nową → Wyślij* → oba zipy z [Releases](../../releases) → Włącz
3. Polska wersja **działa od razu** (gotowe tłumaczenia jadą w paczce). Klucz API
   (*Ustawienia → PNB Auto PL*) podłącz po to, żeby TWOJE zmiany tłumaczyły się same.

⚠️ Jeśli na stronie jest WPML — wyłącz go przed włączeniem Polskiej Wersji (szczegóły w instrukcji).

## Wersje

Gotowe paczki .zip do pobrania: zakładka **Releases**. Historia zmian: `CHANGELOG.md`.
